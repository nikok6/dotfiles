# dotfiles

Personal dotfiles for Claude Code configuration.

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/nikok6/dotfiles/main/install.sh | bash
```

Or clone and run:

```bash
git clone https://github.com/nikok6/dotfiles.git
cd dotfiles
./install.sh
```

The install script will:
1. Download the [claude-statusline](https://github.com/nikok6/claude-statusline) binary for your platform
2. Configure Claude Code settings

Supported platforms: macOS ARM64, Linux x64, Linux ARM64
